Concept Summary

The GSM aggregates semantic triangles across rising/falling classes. Triangle area approximates energy; distributional views (violins) reveal concentration, skew, and tails. Cluster bars summarize event magnitudes per period and help compare windows (Main vs Reference). Giant triangle overlays provide an intuitive comparison of directional energy.

  • Rising vs Falling: semantic classes with domain-specific polarity (e.g., Rising = elevated flood pressure).
  • Distributional risk: tail-aware violin plots capture extreme episodes and regime shifts.
  • Window comparison: select calendar or slider ranges for Main and Reference periods.
